My Tetris 

Objectif : 
To code a Tetris game in Python.

To play the game, you have to install python : https://www.python.org/downloads/
Then just press new game button.

Arrow left and right to move on the left and the right.
And "a" to rotate the brick on the left, "e" to rotate on the right.

If you don't exit before losing, you can save your score, and you have a tab in the main menu to go see them.
"Clear" button delete all saved scores.

Have Fun !
